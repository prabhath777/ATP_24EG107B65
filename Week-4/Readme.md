# Topics Learned

## 1. HTML Document Structure

Learned the basic structure of an HTML document.

```html
<!DOCTYPE html>
<html>
<head>
    <title>My Webpage</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>
```

### Key Tags
- `<html>`
- `<head>`
- `<title>`
- `<body>`

---

## 2. Headings and Paragraphs

Used heading and paragraph tags to organize content.

```html
<h1>Main Heading</h1>
<h2>Sub Heading</h2>

<p>This is a paragraph.</p>
```

# 6. Tables

Learned how to organize structured data using tables.

```html
<table border="1">
    <tr>
        <th>Name</th>
        <th>Role</th>
    </tr>

    <tr>
        <td>Prabhath</td>
        <td>Student</td>
    </tr>
</table>
```

### Tags Used
- `<table>`
- `<tr>`
- `<th>`
- `<td>`

---

# 7. Forms

Learned how forms collect user input.

### Topics Practiced
- Text input
- Password fields
- Email input
- Radio buttons
- Checkboxes
- Dropdown menus
- Textareas
- Submit buttons

### Example Form

```html
<form>

    <label>Name:</label>

    <br><br>

    <br><br>

    <input type="submit">
```

# 8. Semantic HTML

Learned semantic elements for better structure and readability.

### Semantic Tags
- `<header>`
- `<nav>`
- `<main>`
- `<section>`
- `<article>`
- `<footer>`

### Example

```html
<header>
    <h1>My Website</h1>
</header>

<main>
    <section>
        <p>Welcome to my webpage.</p>
    </section>
</main>
```
